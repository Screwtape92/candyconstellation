# Candy Constellation: Space Dodger

A web game built for AgileBridge's company birthday beerfest — an endless space dodger set in an alternate universe made of candy, where astronauts gather candy ingredients to brew the beer.

See `docs/game-design.md`, `docs/architecture.md`, and `docs/asset-spec.md` for
the current game design, architecture, and asset specs, and `CLAUDE.md` for
how this repo's agents/skills are structured. `docs/planning-log.md` records
the reasoning behind the decisions in those docs.
